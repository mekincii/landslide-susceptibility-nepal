# Data

This repository does not include the full raw and processed geospatial datasets due to file size and external data licensing/distribution considerations.

## External data used

- NASA Global Landslide Catalog (GLC)
- SRTM DEM
- CHELSA precipitation data (`precip_bio12_chelsa.tif`)

## Notes

The pipeline scripts in `scripts/` are used to prepare, process, and model the data.

Only lightweight metadata and experiment outputs are included in the repository. Raw rasters, large intermediate files, and trained models are excluded from version control.
